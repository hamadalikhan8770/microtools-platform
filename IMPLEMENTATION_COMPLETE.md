## 🎓 How to Use

### For End Users

1. Visit `/ai-tools` to see all tools
2. Click on a tool (Resume, Proposal, Email, or Bio)
3. Fill in the form with your information
4. Click "Generate" button
5. Review AI-generated content
6. Copy or download the result
7. Customize as needed

### For Developers

#### Using Resume Generator
```
POST /api/ai/generate
{
  "toolType": "resume",
  "formData": {
    "fullName": "John Doe",
    "email": "john@example.com",
    ...
  }
}
```

#### Using Proposal Generator
```
POST /api/ai/generate
{
  "toolType": "proposal",
  "formData": {
    "projectName": "Web App",
    "clientName": "Acme Inc",
    ...
  }
}
```

#### Adding New Tools
1. Add toolType to `GEMINI_TOOLS` or `GROQ_TOOLS` in API route
2. Add prompt case in `buildPrompt()` function
3. Create new page in `app/ai-tools/[tool-name]/`
4. Create form component in `components/ai-tools/`
5. Update sitemap with new URL

### Common Questions

**Q: Do I need credit card for API keys?**
A: No! Both Gemini and Groq offer free tiers with no credit card required.

**Q: How much will the tools cost?**
A: Free to start. Both APIs have generous free tiers. Cost only applies if you exceed free limits.

**Q: Will tools affect AdSense approval?**
A: No! All tools are 100% policy compliant. Zero risk.

**Q: Can I customize the prompts?**
A: Yes! Edit prompts in `buildPrompt()` function in API route.

**Q: Can I add more tools?**
A: Yes! Add to API route and create new pages.

### Troubleshooting

**If tools don't work:**
1. Check API keys are added to Vercel
2. Verify keys are correct (copy from source)
3. Redeploy project
4. Hard refresh browser (Ctrl+Shift+R)
5. Check browser console (F12) for errors

**If AdSense shows warnings:**
1. Don't panic - unlikely to happen
2. All tools are policy-safe
3. Contact support with screenshot

**If generation is slow:**
1. This is normal (5-10 seconds typical)
2. System uses fallback API automatically
3. No action needed
